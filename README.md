# GUTS
Project for GUTS

## Creating maps
Use the following characters in the maps. 
New lines denote new rows and each character denotes a new column.
```
  SPACE (simply a spacebar)
─ HORIZONTAL_LINE
│ VERTICAL_LINE
┌ TOP_LEFT_CORNER
┐ TOP_RIGHT_CORNER
└ BOTTOM_LEFT_CORNER
┘ BOTTOM_RIGHT_CORNER
├ VERTICAL_AND_RIGHT
┤ VERTICAL_AND_LEFT
┴ HORIZONTAL_AND_UP
┬ HORIZONTAL_AND_DOWN
┼ VERTICAL_AND_HORIZONTAL
```

### Example map
```
┌────┬─────────┐
│    │         │
│    └─────┐   │
│          │   │
│  │   │   │   │
│  │   │   │   │
└──┴───┴───┴───┘
```
